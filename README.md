Game Hub é uma plataforma criada para ajudar gamers a organizarem sua vida nos jogos de forma simples, visual e personalizada.

Sabe quando você quer lembrar quais jogos já zerou, quais ainda quer jogar, ou até montar uma lista com os títulos que pretende finalizar no próximo ano? O Game Hub torna isso possível.

No site, o usuário pode:

 • descobrir os jogos mais recomendados e populares do momento

 • criar suas próprias listas personalizadas

 • adicionar jogos e organizá-los por ordem (1º, 2º, 3º lugar, etc.)

 • salvar tudo para acompanhar seu progresso

O objetivo do Game Hub é trazer organização e motivação para o universo gamer, permitindo que cada pessoa gerencie seus games do jeito que quiser — seja para controlar o backlog, registrar conquistas ou planejar o que jogar no futuro.

Game Hub: organize seus jogos, acompanhe suas conquistas e transforme sua experiência gamer.
